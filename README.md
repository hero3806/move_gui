# HUD/GUI Customizer
![HUD/GUI Customizer](https://i.ibb.co/Cp8VFyCY/Untitled-67.png)

Ever wanted to customize your HUD? Well, now you can do that with this mod! (more options coming soon!)
(NOTE: This mod is currently in ALPHA!)

## Features

- Move every little HUD item (eg, hotbar, hearts, armor icon etc.) without any worries!
- 2 Presets to choose from (more coming soon)
- Move your chat window and even change the allignment to the right side!

## Installation

1. Download the latest release from [Modrinth](https://modrinth.com/project/ikI75p6f).
2. Place the `.jar` file in your server's `mods` folder **[FABRIC ONLY]**.
3. Start your minecraft client!
