# HUD/GUI Mover
![Minecord Logo](https://i.ibb.co/Z1BtJ8dx/Untitled-46.png)

A way to communicate in-game to Discord and vice-versa for Fabric!
(NOTE: This mod is currently in BETA!)

## Features

- Seamless chat synchronization between Minecraft and Discord
- Easy configuration

## Installation

1. Download the latest release from [Modrinth](https://modrinth.com/project/g74PXQMz).
2. Place the `.jar` file in your server's `mods` folder.
3. Restart your server.

## Configuration

1. Edit the `config.toml` file generated in the `config` directory.
2. Set your Discord bot token and channel ID.

## Usage

- Every message from Discord will be sent to Minecraft and the vice-versa!
